## Инструкция по запуску

### Предварительные требования
- Установленный Python 3.11+
- Установленный Redis (для задания 6.4)
- Установленный `uv` (менеджер зависимостей)

### Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd ivanov_ii_async

# Создание виртуального окружения
uv venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate  # Windows

# Установка зависимостей
uv add fastapi uvicorn httpx pydantic-settings redis aioredis loguru

# Запуск Redis (в отдельном терминале)
redis-server

# Запуск приложения
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
